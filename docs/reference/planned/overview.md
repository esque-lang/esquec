# Planned features

This section documents features that are designed-but-not-yet-built
(or not-yet-finished) in the reference compiler. Each page
describes:

- **What the feature is** — design intent.
- **Why it is not implemented today** — technical or scope reason.
- **Where it sits in the version plan** — link to the
  [roadmap](roadmap).

These pages are intentionally separated from the
[main spec](/reference/spec/overview) so a reader can see at a
glance what is real versus what is signposted. Newcomers to the
[Learn](/) track should not encounter these pages by accident.
